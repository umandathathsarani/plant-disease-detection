## [1.0.0] - 2026-06-21

### Added

* Initial release of Plant Disease Detection & Diagnosis System.
* ResNet18-based plant disease classification model.
* Transfer learning pipeline using PyTorch.
* FastAPI backend for real-time inference.
* Interactive frontend built with HTML, CSS, and JavaScript.
* Disease treatment recommendation system.
* Hogwarts-themed user interface.
* Gryffindor theme.
* Slytherin theme.
* Ravenclaw theme.
* Hufflepuff theme.
* Image upload and prediction workflow.
* Complete project documentation.

### Machine Learning

* Data exploration pipeline.
* Data preprocessing pipeline.
* Data augmentation techniques.
* Model training workflow.
* Real-time inference testing notebook.
